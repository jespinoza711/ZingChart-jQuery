ZingChart jQuery Table-to-Chart Plugin

Basic Info

This plugin takes the data from an HTML table and converts renders it in a ZingChart. It allows the user to submit JSON data along with the table when rendering, or to pull data from HTML5 custom data attributes.

note: the JS code snippets below are assumed to be within a jQuery $(document).ready() block

Configuration

    Using the plugin requires you include the following scripts:
  • jQuery (ex: <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>)
  • the ZingChart core library scripts (ex: <script src="zingchart/html5_scripts/zingchart-html5-min.js"></script>)
  • the ZingChart jQuery plugin, which this plugin utilizes (ex: <script src="zingchart.jquery.js"></script>)
  • the script for this plugin, zingify.jquery.js (ex: <script src="zingify.jquery.js"></script>)

Basic usage

Converts a (well-formed) HTML table into a ZingChart rendering. For an example of the most basic usage:

A table in the HTML that looks like this:

Sales
month apples oranges
Jan 42 48
Feb 37 52
Mar 48 59
April 36 61
May 35 38

When zingified as follows:

$('table#myTable').zingify();
